# Random Tarot Icon Landing Page

This project is a lightweight landing page that fetches a random tarot icon, description, and attribution from a Google Sheet via Google Apps Script.

## Setup
1. Create a Google Sheet with columns: `title`, `body_text`, `image_url`, `attribution`
2. Deploy the Google Apps Script (Web App)
3. Replace `YOUR_APPS_SCRIPT_URL` in `index.html` with your Web App URL
4. Host the HTML on GitHub Pages or another static host

## Attribution
Icons are used with proper attribution in the sheet column `attribution`.
